📌 About This Project

StreamVerse is an academic/demo project created by Payal, Diksha, Latika, and Smily. It is not a real commercial streaming service. All content, images, and data used on this platform are for educational and demonstration purposes only.

This project showcases a full-featured OTT platform UI built entirely with HTML, CSS, and JavaScript — including movie sections, TV shows, live sports, and inner pages like this one.

📋 Information We Collect

As this is a demo project, no real user data is collected, stored, or transmitted. Any forms on this website are for UI demonstration only and do not submit data to any server.

  • No personal data is stored or processed
  • No payment information is accepted or processed
  • No cookies beyond standard browser defaults are set
  • No analytics or tracking scripts are embedded

🛠️ How We Use Information

Since no data is collected, no information is used for any purpose. In a real production version of StreamVerse, data would be used only to personalise the user experience, process subscriptions, and improve platform quality.

🔐 Data Security

No user data is stored in this project. In a real deployment, StreamVerse would implement 256-bit SSL encryption, regular security audits, and strict access controls to protect all user information.
